# Cursor Setup & AI Tooling Project

## Overview
This project demonstrates my ability to independently set up a development environment, integrate AI tools, and document the process clearly. The objective was to install required tools, configure them correctly, and push the setup to a public GitHub repository.

## Tools & Technologies Used
- **Cursor IDE** – AI-powered code editor
- **Claude Code Extension (Anthropic)** – AI coding assistant
- **Codex Extension** – Additional AI support for development
- **GitHub** – Version control and repository hosting

## Steps Completed

### 1. Environment Setup
- Downloaded and installed Cursor IDE
- Verified installation and basic functionality

### 2. Extension Installation
- Installed **Claude Code (official extension by Anthropic)**
- Installed **Codex extension**
- Logged into both tools and ensured proper integration

### 3. GitHub Setup
- Created a public repository
- Configured repository for version control

### 4. Repository Integration
- Cloned the repository into Cursor IDE
- Verified local and remote sync

### 5. Documentation
- Created this `README.md` file
- Structured the documentation for clarity and readability

### 6. Version Control
- Staged changes
- Committed with appropriate message
- Pushed to GitHub successfully

## Challenges Faced & Solutions

### Issue 1: Multiple Claude Extensions Available
- **Challenge:** Several similar extensions appeared in the marketplace
- **Solution:** Selected the official extension published by **Anthropic** to ensure authenticity

### Issue 2: Initial Confusion in Git Workflow
- **Challenge:** Understanding clone → commit → push sequence
- **Solution:** Referred to documentation and followed step-by-step Git workflow

### Issue 3: Extension Login & Setup
- **Challenge:** Authentication flow was unclear initially
- **Solution:** Followed prompts and completed login to activate features
